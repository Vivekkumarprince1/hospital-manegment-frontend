# Hospital Management System Frontend

A modern hospital management system built with React, Vite, and Tailwind CSS.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Technologies Used
- React
- Vite
- Tailwind CSS
- Material UI
- React Router
- Axios
